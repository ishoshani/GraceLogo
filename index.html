Grace Logo Dialect Docs

Welcome to a beginners guide to using Logo in Grace!

This is a simple dialect that allows easy access to turtle graphics

We choose to keep things simple first of all by restricting the kind of variables e can declare(and we do have to declare them as a type)

Number like Var i : Number := 50

Boolean like Var i : Boolean := true

String like Var i : String := "Hello World"

Next we consider the attributes for our turtle

location a Point such as 250@250
heading a Number in degrees such as 270
nib a Boolean such as true for knowing if we are currently drawing

we control these using the following methods:

right(degree:Number) and left(degree:Number) turn us one way or the other
setAngle(degree:Number) lets us set an angle of our choice starting with right along the x-axis as 0

moveForward(units:Number) moves our turtle a number of units in its current direction, the heading
nibSwitch alternates whether our pen is on or off
setNib(isOn:Boolean)) lets us manually set the nib to On as true or Off as false

before we do anything, we want to start up the picture! So use show for that!

Make sure popups are allowed though, and that you have the objectdraw library present!

An Example to get us started!
--> 
dialect "logoDialect"
show // creates our 500 by 500 window, and starts the turtle at 250,250, with initial heading 270
right(90)
nibSwitch //turns on the nib, since it starts at off
moveForward(50)
nibSwitch
right(90)
moveForward(50)
left(90)
nibSwitch
MoveForward(50)
